# Command interpreter

## Installation

Clone this repositary:

    $ git clone https://github.com/hollaze/AirBnB_clone.git 

Access the folder:

    $ cd AirBnB_clone

You can access the console via two modes.

Interactive mode:

    $ ./console.py

Non-interactive mode:

    $ echo "<command>" | ./console.py
